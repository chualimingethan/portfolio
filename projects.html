OpenCV PI and Ardiuno Robot

2026

Built a system that uses a Raspberry Pi running OpenCV and YOLO to identify objects and communicate with an Arduino through USB serial. The model was trained on a custom dataset with augmentation to improve robustness and reduce overfitting. The Raspberry Pi runs the vision model and communicates detection results to the Arduino for the folowing action that need to be executed. For example, in the first task, the Raspberry Pi detects the object and sends a command to the Arduino, which then controls the motors to retrieve the correct object. I mainly worked on dataset collection, model training, testing, writing of the python scripts, and system integration.

Python C++ YOLO OpenCV Raspberry Pi Arduino Serial (UART)
