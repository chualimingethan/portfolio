OpenCV PI and Ardiuno Robot

Built a system using OpenCV to detect objects and send commands via USB serial to an Arduino-controlled robot. The model was trained on a custom dataset with augmentation to improve robustness and reduce overfitting. The Raspberry Pi runs the vision model and communicates detection results to the Arduino for motor control.

OpenCV Photobooth

Developed a facial landmark-based photobooth that applies real-time effects and background replacement.
